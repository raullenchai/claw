# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-17

### Added
- Initial release of Claw
- Live terminal view with real-time tmux pane content
- Session, window, and pane selection
- Quick action buttons (yes, no, continue, /compact, Ctrl+C)
- Git status monitoring (branch, changes, recent commits)
- System stats (CPU, memory, load)
- Claude process monitoring
- Mobile-first responsive design
- Pull-to-refresh gesture support
- Auto-refresh with configurable interval
- Dark theme optimized for OLED screens
- REST API for programmatic access

### Technical
- Single-file Python application
- Zero external dependencies (stdlib only)
- Works with Python 3.8+
